## CSS Implementation Types

This project demonstrates the different ways to apply CSS in HTML.

## Types of CSS Used
1. Inline CSS
   - CSS is applied directly inside an HTML tag using the style attribute.

2. Internal CSS
   - CSS is written inside the <style> tag within the <head> section of the HTML file.

3. External CSS
   - CSS is written in a separate .css file and linked to the HTML file using the <link> tag.

## Project Files
- index.html → Main HTML file
- style.css → External CSS file

## Purpose of the Project
The purpose of this project is to understand how CSS works and how styles can be applied to HTML elements using different methods.

## Technologies Used
- HTML5
- CSS3

## Learning Outcome
By completing this project, you will understand:
- The concept of Cascading Style Sheets
- Different ways to implement CSS
- Basic styling of HTML elements
